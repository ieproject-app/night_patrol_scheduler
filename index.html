📤 Ekspor ke Excel

📋 Cara menggunakan di Excel (Text to Columns):
  1. Salin seluruh isi dari kotak teks di atas dengan klik tombol "📋 Salin Teks"
  2. Buka Excel dan tempel (Paste) di sel A1 pada lembar kerja baru
  3. Pilih seluruh kolom A, lalu pergi ke menu Data → Text to Columns
  4. Pilih Delimited, lalu klik Next
  5. Beri centang pada Comma (koma) dan hilangkan centang lainnya
  6. Klik Finish untuk menyelesaikan proses
🖨️ Tips untuk Screenshot/Print:
  1. Klik tombol "🖨️ Print/Screenshot" untuk membuka tampilan yang dioptimalkan
  2. Gunakan Ctrl+P (Windows) atau Cmd+P (Mac) untuk print
  3. Untuk screenshot, gunakan aplikasi screenshot favorit Anda
  4. Layout sudah dioptimalkan untuk ukuran kertas standar
